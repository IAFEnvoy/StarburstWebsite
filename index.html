About Us

Started as a rewrite version of Abyss, now Starburst Overlay is one of the most excellent Hypixel stats search softwares.

  • Ingame Stats Searcher
  • All Stats Searcher
  • Other Features

The software will automatically read your game's log and identify the players in current game. Then it will download the stats data from APIs and parse them to the window.

Faster

Use original APIs to fetch data. Also cache all these data. (Permanently for Mojang API data, 2 minutes for Hypixel API data)

More Game Mode

Support Bedwars, Sky War, Murder Mystery, Duel, UHC, Mega Wall, Wool War, Blitz Survival Game and all their sub-modes.

Stats Sort

You can sort any stat by clicking the header of the table.

Blacklist System

Show blacklist player by using red tags.

Open an extra window to search all stats of a player.

More Details

Show more stats and game types than in the ingame one.

    Frequently Asked Questions

    Go to developer.hypixel.net to get the key.

    Firstly, run /who command ingame. If still nothing happened, check you language. The Hypixel ingame language should be the same as the software language. Also check your log path.

    If all these solutions don't work, report it to the developer.

    There is a request limit in Mojang API. When reach the limit, the software will auto-retry to get data. This operation will take some time.

    Also, your network environment is an important factor to cause this.
